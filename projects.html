ID3 Meta Tag Editor

This is a simple, feature packed ID3 Tag editor for editing metadata on mp3 files. it allows for you to edit an entire directory with ease. You can edit the title, artist, album, year and song name. Another feature is that it allows you to wipe all meta data from any mp3 files currently saved in its cache.

Tiff 2 PDF

Tiff 2 PDF is a simple tool in c# for converting Tiff files into PDF Files. To use it you simply select the directory with the .tiff files in it and then click convert. It has a simple output box to give feedback based on its current operation.
